# 云脑智能体平台 - 前端UI框架布局设计方案

## 一、整体布局架构

### 1.1 布局模式
采用**左侧固定导航 + 右侧主内容区**的经典SaaS应用布局模式，适合智能体管理和任务处理等复杂功能场景。

### 1.2 布局结构
```
┌─────────────────────────────────────────────────────────────┐
│                          顶部状态栏 (可选)                      │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                  │
│          │                                                  │
│          │                                                  │
│  左侧    │              主内容区                              │
│  主导航  │        (智能体详情、任务列表、任务详情等)              │
│          │                                                  │
│          │                                                  │
│          │                                                  │
│          │                                                  │
└──────────┴──────────────────────────────────────────────────┘
```

## 二、左侧主导航设计

### 2.1 导航区域布局
**尺寸规格：**
- 宽度：240px（展开状态）/ 64px（折叠状态）
- 高度：100vh（全屏高度）
- 位置：fixed（固定定位）
- 层级：z-index: 1000

**视觉设计：**
- 背景色：深色系（#1a1a1a 或 #2d2d2d）体现专业科技感
- 文字颜色：浅色系（#ffffff / #e0e0e0）确保对比度
- 边框：右侧 1px 边框线，分隔导航与内容区

### 2.2 导航层级结构

#### 第一层：品牌区域（顶部固定）
```
┌─────────────────────────┐
│   [Logo] 云脑智能体      │  ← 品牌标识 + 平台名称
│   [折叠按钮]             │  ← 支持展开/折叠导航
└─────────────────────────┘
```
- 高度：64px
- Logo + 平台名称（展开时）/ 仅Logo（折叠时）
- 可点击回到首页

#### 第二层：全局导航（中部滚动区域）
**导航菜单项：**

1. **首页**
   - 图标：Home
   - 功能：平台概览、数据统计、快速入口
   - 路由：`/`

2. **我的智能体**
   - 图标：Robot / Brain
   - 功能：已订阅/已创建的智能体列表
   - 路由：`/agents`
   - 子级：
     - 全部智能体
     - 我的订阅
     - 我创建的

3. **智能体市场**
   - 图标：Store / Grid
   - 功能：浏览和订阅可用的智能体
   - 路由：`/marketplace`
   - 子级（可选）：
     - 代码审计
     - 漏洞挖掘
     - 渗透测试
     - 知识问答

4. **资讯中心** (从原有门户继承)
   - 图标：Newspaper
   - 功能：安全资讯浏览
   - 路由：`/news`

5. **论文库** (从原有门户继承)
   - 图标：BookOpen
   - 功能：学术论文查阅
   - 路由：`/papers`

6. **资源中心** (从原有门户继承)
   - 图标：Archive
   - 功能：安全工具和资源导航
   - 路由：`/resources`

#### 第三层：底部导航（底部固定）

**功能入口：**

1. **📐 UI模板**
   - 图标：📐
   - 功能：查看智能体详情页面的前端布局模板
   - 路由：`/templates.html`
   - 用途：开发人员查看和预览各种UI布局模板
   - 说明：展示快速对话、任务管理、知识库等所有模板页面

2. **💬 用户反馈**
   - 图标：💬
   - 功能：提交用户反馈和建议
   - 路由：待定

3. **❓ 帮助中心**
   - 图标：❓
   - 功能：使用帮助和文档
   - 路由：待定

4. **用户区域**
   ```
   ┌─────────────────────────┐
   │  [用户头像] 用户名        │  ← 用户信息
   │  ├─ 个人中心              │  ← 下拉菜单
   │  ├─ 账户设置              │
   │  └─ 退出登录              │
   └─────────────────────────┘
   ```
   - 高度：64px
   - 点击展开下拉菜单
   - 快捷访问个人中心和设置

### 2.3 导航交互设计

**状态反馈：**
- 当前激活项：高亮背景色（如深蓝色 #1890ff 或品牌色）
- 悬停状态：轻微背景色变化
- 点击反馈：涟漪动画效果

**展开/折叠：**
- 默认展开（240px）
- 折叠后仅显示图标（64px）
- 支持鼠标悬停时临时展开（Hover to expand）
- 折叠状态通过localStorage持久化

**子菜单展开：**
- 点击父级展开子菜单
- 使用缩进和连接线显示层级关系
- 折叠状态下，子菜单以悬浮菜单形式显示

## 三、主内容区设计

### 3.1 内容区布局
**尺寸规格：**
- 宽度：calc(100vw - 240px)（导航展开）/ calc(100vw - 64px)（导航折叠）
- 高度：100vh
- 内边距：24px（桌面端）/ 16px（移动端）
- 背景色：浅色系（#f5f5f5 / #fafafa）

**区域划分：**
```
┌──────────────────────────────────────┐
│  面包屑导航                            │
├──────────────────────────────────────┤
│                                      │
│  页面标题 + 操作按钮区域                │
│                                      │
├──────────────────────────────────────┤
│                                      │
│                                      │
│          主要内容区域                  │
│     (根据不同页面展示不同内容)          │
│                                      │
│                                      │
└──────────────────────────────────────┘
```

## 四、智能体详情页面设计

### 4.1 页面结构
当用户点击某个智能体时，进入智能体详情页面：

**路由：** `/agents/:agentId`

**页面布局：**
```
┌─────────────────────────────────────────────────────┐
│  首页 > 我的智能体 > [智能体名称]                      │ ← 面包屑
├─────────────────────────────────────────────────────┤
│                                                     │
│  [智能体图标]  智能体名称              [订阅/取消订阅]  │ ← 页头区域
│  智能体简介描述...                                   │
│  标签：代码审计 | Python | 高级                      │
│                                                     │
├──────────┬──────────────────────────────────────────┤
│          │                                          │
│  侧边栏   │          任务列表/任务详情                 │ ← 核心功能区
│  (导航)  │                                          │
│          │                                          │
└──────────┴──────────────────────────────────────────┘
```

### 4.2 智能体详情 - 左侧侧边栏（二级导航）

**位置：** 主内容区内的左侧
**宽度：** 200px（可折叠为图标栏）

**设计原则：**
- 突出核心功能，减少层级
- 常用操作优先展示
- 清晰的视觉分组

**导航菜单项：**

#### 第一组：核心功能区

1. **💬 快速对话**
   - 功能：直接在当前页面启动新对话，无需跳转
   - 路由：`/agents/:agentId/chat`
   - 默认激活项
   - 样式：使用主色调按钮突出显示
   - 说明：点击后在右侧主内容区打开对话界面

2. **📋 任务管理**
   - 功能：查看和管理所有历史任务
   - 路由：`/agents/:agentId/tasks`
   - 子功能：
     - 全部任务
     - 对话式任务
     - GUI交互式任务
     - 已完成 / 进行中 / 失败
   - 显示未完成任务数量角标

#### 第二组：资源管理区

3. **📚 知识库**
   - 功能：管理智能体使用的知识库和文档
   - 路由：`/agents/:agentId/knowledge`
   - 子功能：
     - 文档管理（上传、删除、编辑）
     - 知识库同步状态
     - 向量化进度
   - 显示知识库文档数量

4. **🔧 工具插件**
   - 功能：管理智能体可用的工具、API和插件
   - 路由：`/agents/:agentId/tools`
   - 子功能：
     - 已安装工具列表
     - 工具市场（安装新工具）
     - 工具配置和授权
   - 显示已启用工具数量

#### 第三组：配置和分析区

5. **📊 数据统计**
   - 功能：任务执行统计、使用情况分析
   - 路由：`/agents/:agentId/analytics`
   - 子功能：
     - 任务执行趋势图
     - 成功率统计
     - Token消耗统计
     - 响应时间分析
   - 显示关键指标（如今日任务数）

6. **⚙️ 配置设置**
   - 功能：智能体参数配置和个性化设置
   - 路由：`/agents/:agentId/settings`
   - 子功能：
     - 基础信息（名称、描述、图标）
     - 模型配置（模型选择、温度、Token限制）
     - 提示词设置（系统提示词、角色设定）
     - 高级配置（API密钥、回调设置）
   - 需保存/放弃更改确认

#### 可选功能（根据智能体类型显示）

7. **📝 提示词模板** (可选)
   - 功能：预设的常用提示词模板库
   - 路由：`/agents/:agentId/templates`
   - 子功能：
     - 模板列表
     - 新建/编辑模板
     - 模板分类管理
   - 适用场景：需要频繁使用固定提示词的智能体

8. **🔗 集成管理** (可选)
   - 功能：第三方服务集成（Webhook、API等）
   - 路由：`/agents/:agentId/integrations`
   - 适用场景：需要对接外部系统的智能体

#### 底部操作区

9. **分享协作** (底部固定)
   - 按钮样式：次要按钮
   - 功能：分享智能体给团队成员
   - 弹出模态框进行分享设置

### 4.3 智能体详情 - 右侧主内容区

**位置：** 侧边栏右侧
**宽度：** calc(100% - 200px - 48px)

根据左侧侧边栏选中的菜单项，展示不同内容：

## 五、任务详情页面设计

### 5.1 对话式任务详情页面

**路由：** `/agents/:agentId/tasks/:taskId`
**任务类型：** 对话式 (Chat-based)

**页面布局：**
```
┌─────────────────────────────────────────────────────┐
│  任务 #001 - 代码安全审计                      [...]  │ ← 任务标题 + 操作菜单
│  类型：对话式 | 状态：进行中 | 创建于 2024-12-09      │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌───────────────────────────────────────────┐     │
│  │ 用户: 请帮我审计这段Python代码的安全性...   │     │ ← 对话气泡（用户）
│  │ 2024-12-09 10:30                          │     │
│  └───────────────────────────────────────────┘     │
│                                                     │
│  ┌───────────────────────────────────────────┐     │
│  │ 智能体: 我已经分析了您的代码，发现以下      │     │ ← 对话气泡（AI）
│  │ 潜在安全问题：                             │     │
│  │ 1. SQL注入风险...                         │     │
│  │ 2. XSS漏洞...                             │     │
│  │ [代码块展示]                               │     │
│  │ 2024-12-09 10:32                          │     │
│  └───────────────────────────────────────────┘     │
│                                                     │
│  ┌───────────────────────────────────────────┐     │
│  │ 用户: 如何修复SQL注入问题？                │     │
│  │ 2024-12-09 10:35                          │     │
│  └───────────────────────────────────────────┘     │
│                                                     │
│  ...                                                │
│                                                     │
├─────────────────────────────────────────────────────┤
│  [📎] [输入框: 输入您的问题...]          [发送]      │ ← 输入区域
└─────────────────────────────────────────────────────┘
```

**核心元素：**

1. **任务头部信息栏**
   - 任务ID、标题
   - 任务类型、状态标签
   - 创建时间
   - 操作菜单（...）：重命名、导出、删除等

2. **对话消息区域**
   - 用户消息：右对齐，浅色背景
   - AI消息：左对齐，深色背景或品牌色背景
   - 消息时间戳
   - 支持Markdown渲染（代码块、列表、粗体等）
   - 支持代码高亮显示
   - 消息操作：复制、点赞、反馈

3. **输入区域**
   - 多行文本输入框
   - 附件上传按钮（上传代码文件等）
   - 发送按钮
   - 支持快捷键：Shift+Enter 发送
   - 输入提示/建议（可选）

**交互特性：**
- 自动滚动到最新消息
- 消息加载中的动画效果（打字效果）
- 长消息支持展开/折叠
- 代码块支持一键复制
- 支持消息搜索功能

### 5.2 GUI交互式任务详情页面

**路由：** `/agents/:agentId/tasks/:taskId`
**任务类型：** GUI交互式 (GUI-based)

**页面布局：**
```
┌─────────────────────────────────────────────────────┐
│  任务 #002 - 渗透测试辅助                      [...]  │ ← 任务标题 + 操作菜单
│  类型：GUI交互式 | 状态：进行中 | 创建于 2024-12-09    │
├──────────┬──────────────────────────────────────────┤
│          │                                          │
│  左侧    │          主操作区域                        │
│  对话    │      (可视化工具、表单、结果展示等)          │
│  面板    │                                          │
│          │                                          │
│  (可选)  │                                          │
│          │                                          │
└──────────┴──────────────────────────────────────────┘
```

**布局说明：**

GUI交互式任务采用**分屏布局**，适合需要复杂交互和可视化展示的场景。

#### 左侧对话面板（可选，宽度：30-40%）
- 与智能体进行对话式引导
- 显示操作提示和建议
- 展示执行进度和日志
- 可折叠/展开，默认展开

#### 右侧主操作区域（宽度：60-70%）
根据智能体功能的不同，展示不同的GUI组件：

**示例1：代码审计工具**
```
┌─────────────────────────────────────────────┐
│  [上传文件] [选择目录]                        │ ← 工具栏
├─────────────────────────────────────────────┤
│                                             │
│  代码编辑器                                  │ ← 代码编辑器
│  (语法高亮、行号、折叠功能)                   │
│                                             │
├─────────────────────────────────────────────┤
│  审计结果：                                  │
│  ┌─────────────────────────────────────┐   │
│  │ ⚠️ 高危: SQL注入 (行 23)             │   │ ← 结果列表
│  │ ⚠️ 中危: XSS漏洞 (行 45)             │   │
│  │ ℹ️ 低危: 硬编码密钥 (行 67)           │   │
│  └─────────────────────────────────────┘   │
│                                             │
└─────────────────────────────────────────────┘
```

**示例2：渗透测试工具**
```
┌─────────────────────────────────────────────┐

└─────────────────────────────────────────────┘
```

**核心元素：**

1. **工具栏/配置区**
   - 操作按钮（上传、下载、执行等）
   - 配置选项（下拉框、开关等）
   - 快捷操作入口

2. **可视化操作区**
   - 代码编辑器
   - 表单输入
   - 图表展示
   - 树形结构
   - 拖拽上传区域

3. **结果展示区**
   - 执行结果列表
   - 数据表格
   - 图表可视化
   - 日志输出
   - 导出功能

4. **左侧对话面板（可选）**
   - 智能体引导和提示
   - 操作建议
   - 进度反馈

**交互特性：**
- 实时反馈：操作立即响应，显示加载状态
- 数据校验：表单输入实时校验
- 进度提示：长时间操作显示进度条
- 结果导出：支持导出为PDF、Excel等格式
- 快捷键支持：常用操作提供快捷键

### 5.3 两种任务类型的切换和融合

**设计考虑：**

部分智能体可能同时支持对话式和GUI交互式两种模式，用户可以根据需求灵活切换。

**切换方式：**

在任务详情页面顶部提供**模式切换器**：
```
┌─────────────────────────────────────────────────────┐
│  任务 #003 - 代码审计                         [...]  │
│  [💬 对话模式] [🖥️ GUI模式]  ← Tab切换             │
├─────────────────────────────────────────────────────┤
│  (根据选择的模式展示对应的界面)                       │
└─────────────────────────────────────────────────────┘
```

**融合场景：**

对于复杂任务，可以在GUI模式下嵌入对话面板，实现**对话辅助 + 可视化操作**的融合体验：
- 左侧：对话面板（30%宽度）
- 右侧：GUI操作区（70%宽度）
- 用户可以通过对话引导AI，同时在GUI界面进行可视化操作

- 智能体详情左侧侧边栏：改为顶部Tab导航
- GUI交互式任务：垂直堆叠布局
  - 对话面板在上
  - 操作区在下
  - 支持上下滑动切换
- 任务列表改为卡片式堆叠
