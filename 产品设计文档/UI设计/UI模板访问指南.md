## 🎯 访问方式

### 方法1：通过任意页面的左侧导航访问

1. 打开以上任意页面（推荐从 `dashboard.html` 开始）
2. 在页面左侧导航栏滚动到底部
3. 在用户头像上方可以看到：

```
┌─────────────────────────┐
│ 📐 UI模板               │ ← 新增入口！
│ 💬 用户反馈             │
│ ❓ 帮助中心             │
│ ─────────────────────   │
│ [头像] 测试用户         │
└─────────────────────────┘
```

4. 点击 **📐 UI模板** 即可进入模板展示页

## 📋 模板展示页功能

在模板展示页面（`templates.html`）中，你可以：

### 1️⃣ 浏览所有模板

模板按功能分类展示：
- 💬 **对话与任务**：空状态、对话、历史任务
- 📚 **资源管理**：知识库、工具插件
- 📊 **配置分析**：数据统计、配置设置

### 2️⃣ 筛选模板

使用顶部分类按钮筛选：
- **全部模板** - 显示所有模板
- **对话与任务** - 仅显示空状态、对话、历史任务
- **资源管理** - 仅显示知识库、工具插件
- **配置分析** - 仅显示数据统计、配置设置

### 3️⃣ 预览模板

- **悬停**：鼠标悬停在模板卡片上，显示"预览模板"按钮
- **点击**：点击按钮在新窗口打开完整的模板页面
- **查看**：浏览模板的实际效果和交互功能

### 4️⃣ 查看设计说明

页面底部包含详细的设计说明：
- 布局结构
- 侧导航分组
- 交互特性
- 文件列表
- 参考文档链接
