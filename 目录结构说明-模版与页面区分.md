# 目录结构说明 - 模版与页面区分

## 核心原则

**模版（Template）**：通用的、脱敏的UI模版，用于展示界面结构和功能，不包含具体业务内容。

**实际应用页面（Agent Pages）**：基于模版的实际业务应用，包含具体的业务内容和数据。

---

## 目录结构

```
frontend/
├── templates.html                    # UI模版库入口页面

│   │   ├── agent-search.css         # 信息检索页面样式（模版和实际应用共用）
│   │   ├── agent-news-detail.css     # 安全资讯详情页样式
│   │   └── ...
│   │
│   ├── js/                          # 智能体页面专用脚本
│   │   ├── agent-detail.js          # 智能体详情页通用脚本
│   │   ├── agent-search.js          # 信息检索页面脚本（模版和实际应用共用）
│   │   ├── agent-news.js            # 安全资讯列表页脚本
│   │   ├── agent-news-detail.js     # 安全资讯详情页脚本
│   │   └── ...
│   │

│       │

│       ├── agent-detail-search.html          # 信息查看/检索模版（列表页）
│       └── agent-detail-search-detail.html   # 信息查看/检索模版（详情页）
│       │

│   └── ...

    ├── common.js                     # 通用脚本
    ├── dashboard.js                  # 仪表盘脚本
    └── ...
```

## 模版 vs 实际应用对照表

### 信息查看/检索模式

| 类型 | 文件路径 | 用途 | 内容特点 |
|------|---------|------|---------|
| **模版** | `agents/pages/agent-detail-search.html` | UI模版库预览 | 通用示例，已脱敏，不包含安全相关内容 |
| **模版** | `agents/pages/agent-detail-search-detail.html` | UI模版库预览 | 通用示例，已脱敏，不包含安全相关内容 |
| **实际应用** | `agents/pages/agent-news.html` | 安全资讯智能体 | 包含安全资讯的具体内容和数据 |
| **实际应用** | `agents/pages/agent-news-detail.html` | 安全资讯智能体 | 包含安全资讯的具体内容和数据 |

### 快速对话模式

| 类型 | 文件路径 | 用途 | 内容特点 |
|------|---------|------|---------|
| **实际应用** | `agents/pages/agent-detail-chat.html` | 快速对话助手示例 | 展示快速对话模式的界面和功能 |

## 路径引用规则

### 从 `agents/pages/` 目录引用资源

**引用全局资源（frontend根目录）：**
- CSS: `../../css/common.css`
- JS: `../../js/dashboard.js`
- HTML: `../../dashboard.html`, `../../templates.html`

**引用agents目录下的资源：**
- CSS: `../css/agent-detail.css`
- JS: `../js/agent-search.js`

**引用同目录下的页面：**
- HTML: `agent-detail-search-detail.html`（相对路径，无需前缀）

## 入口页面链接配置

### 1. UI模版库（templates.html）

**信息查看/检索模式 - 预览模版按钮：**
```html
<a href="agents/pages/agent-detail-search.html" class="btn btn-primary" target="_blank">
    预览模板
</a>
```
→ 指向：**模版页面**（通用脱敏）

### 2. 智能体仪表盘（dashboard.html）

**安全资讯智能体 - 立即使用按钮：**
```html
<a href="agents/pages/agent-news.html" class="btn btn-primary btn-sm">立即使用</a>
```
→ 指向：**实际应用页面**（安全资讯内容）

## 样式文件使用规则

### 共用样式文件

- `agents/css/agent-search.css` - 信息检索页面样式（模版和实际应用共用）
- `agents/css/agent-detail.css` - 智能体详情页通用样式

### 专用样式文件

- `agents/css/agent-news-detail.css` - 安全资讯详情页专用样式（仅实际应用使用）

### 全局样式文件

- `css/common.css` - 全局通用样式
- `css/dashboard.css` - 仪表盘样式

---

## 注意事项

1. **不要混淆模版和实际应用**
   - 模版页面（`agent-detail-search*.html`）保持通用和脱敏
   - 实际应用页面（`agent-news*.html`）包含具体业务内容

2. **路径引用必须正确**
   - 从 `agents/pages/` 引用全局资源需要 `../../` 前缀
   - 从 `agents/pages/` 引用 `agents/` 下的资源需要 `../` 前缀

3. **样式文件路径**
   - 模版页面和实际应用页面可以共用样式文件
   - 但要注意样式文件路径的正确性

4. **文档更新**
   - 修改目录结构或文件位置时，及时更新此文档
   - 确保所有路径引用都经过验证

## 验证清单

在修改或新增页面时，请检查：

- [ ] 样式文件路径是否正确（`../../css/` 或 `../css/`）
- [ ] 脚本文件路径是否正确（`../../js/` 或 `../js/`）
- [ ] HTML链接路径是否正确（`../../` 或相对路径）
- [ ] 模版页面保持通用和脱敏
- [ ] 实际应用页面包含具体业务内容
- [ ] `templates.html` 中的预览链接指向模版页面
- [ ] `dashboard.html` 中的使用链接指向实际应用页面
