# 云脑门户页前端静态页面

基于PRD和UI设计需求开发的前端静态页面项目。

## 项目结构

```
frontend/
├── index.html              # 首页（云脑模块）
├── news.html               # 资讯列表页
├── news-detail.html        # 资讯详情页
├── papers.html             # 论文列表页
├── paper-detail.html       # 论文详情页
├── resources.html          # 资源页面
├── css/
│   ├── common.css         # 公共样式
│   ├── index.css          # 首页样式
│   ├── news.css           # 资讯列表页样式
│   ├── news-detail.css    # 资讯详情页样式
│   ├── papers.css         # 论文列表页样式
│   ├── paper-detail.css   # 论文详情页样式
│   └── resources.css      # 资源页面样式
├── js/
│   ├── common.js          # 公共JavaScript
│   └── resources.js       # 资源搜索功能
└── README.md              # 项目说明文档
```

## 功能模块

### 1. 云脑模块（首页）
- 平台介绍和使命展示
- 核心价值主张介绍
- 行动召唤按钮（立即开始、试用云脑）
- SEO优化：结构化数据标记、语义化HTML

### 2. 资讯模块
- 资讯列表页：卡片式布局，支持分类筛选
- 资讯详情页：完整内容展示，相关推荐
- 分页功能
- SEO优化：独立URL、元数据优化、内链关联

### 3. 论文模块
- 论文列表页：卡片式布局，支持多维度筛选
- 论文详情页：AI全文总结突出展示、完整论文内容
- 分页功能
- SEO优化：学术论文结构化数据、相关论文推荐

### 4. 资源模块
- 资源分类展示（工具、文档、学习资源）
- 搜索功能
- 网格布局，响应式设计

## 设计特点

### 响应式设计
- 桌面端（≥1200px）：多列布局，充分利用屏幕空间
- 平板端（768px-1199px）：适配中等屏幕，调整列数和间距
- 移动端（<768px）：单列布局，汉堡菜单，触摸优化

### SEO优化
- 语义化HTML标签（header、nav、main、article、section等）
- 清晰的标题层级（h1、h2、h3等）
- 元数据优化（title、description、keywords）
- 结构化数据标记（Schema.org）
- 内链关联设计

### 用户体验
- 清晰的导航结构
- 悬停和点击状态反馈
- 加载状态提示
- 友好的错误处理

## 使用方法

1. 直接在浏览器中打开 `index.html` 即可访问首页
2. 通过导航菜单可以访问各个功能模块
3. 所有页面都是静态HTML，无需服务器即可运行

## 浏览器支持

- Chrome（推荐）
- Firefox
- Safari
- Edge
